# slidev-component-admonition

## Install

```shell
npm install https://gitpkg.now.sh/chick-p/slidev-components/components/admonition?main
```

## Configuration

```markdown
---
addons:
  - slidev-component-admonition
---
```

Or in your `package.json``:

```json
{
  "slidev": {
    "addons": ["slidev-component-admonition"]
  }
}
```
