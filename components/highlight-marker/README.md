# slidev-component-highlight-marker

## Install

```shell
npm install https://gitpkg.now.sh/chick-p/slidev-components/components/highlight-marker?main
```

## Configuration

```markdown
---
addons:
  - slidev-component-highlight-marker
---
```

Or in your `package.json``:

```json
{
  "slidev": {
    "addons": ["slidev-component-highlight-marker"]
  }
}
```
