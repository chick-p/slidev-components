# slidev-theme-clean

## Install

```shell
npm install https://gitpkg.now.sh/chick-p/slidev-components/themes/slidev-theme-clean?main
```

## Configuration

```markdown
---
addons:
  - slidev-theme-clean
---
```

Or in your `package.json``:

```json
{
  "slidev": {
    "addons": ["slidev-theme-clean"]
  }
}
```
